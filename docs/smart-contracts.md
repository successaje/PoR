# Smart Contracts

## Truth Certificate NFT
The final verification mints a Truth Certificate NFT on the Mantle network.

### Metadata
- Verification timestamp
- Consensus score
- Participating agents
- Evidence hashes
- Confidence decay timer

## Dynamic Truth Decay
Truth expires gradually. Confidence decays over time as market conditions evolve, necessitating re-evaluation by the agents. This aligns with ERC-8004 principles for dynamic, evolving assets.
