# UI/UX Direction

## Aesthetic
- Cinematic, Institutional, AI-native, Premium, Futuristic, Dark-mode-first.
- Inspirations: Palantir, Apple visionOS, Interstellar UI, Bloomberg Terminal.

## Core Experience: The Debate Chamber
- Must feel like "Mission control for truth verification."
- Dark futuristic interface with glassmorphism.
- Glowing node connections and live streaming agent panels.

## Animation Details
- **Agent Activation:** Nodes illuminate sequentially.
- **Consensus Formation:** Circular convergence animation.
- **Fraud Detection:** Glitch distortions + red pulses.
- **Confidence Updates:** Smooth radial graph transitions.
- **NFT Minting:** Massive cinematic confirmation sequence.
