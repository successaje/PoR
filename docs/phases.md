# Development Phases

## Phase 1: The Hackathon Build (MVP)
- Monorepo setup (Next.js, FastAPI, Foundry)
- Basic Multi-Agent Verification (mocked APIs for speed and reliability during demo)
- The Debate Chamber visualizer (Live streaming WebSocket)
- Minting Truth Certificate NFT on Mantle testnet

## Phase 2: Alpha Protocol
- Integration with real-world APIs (Google Maps, OpenStreetMap, local registries)
- Implementation of Aletheia Consensus Engine rules
- ERC-8004 dynamic truth decay logic on-chain

## Phase 3: Mainnet Launch
- Fully decentralized node runners for the Agents
- DAO governance for Aletheia parameters
- Expansion beyond Real Estate (Shipping, Insurance, Commodities)
