# Consensus Engine (Aletheia)

Aletheia is the core AI consensus protocol.

## Responsibilities
- Aggregate evidence from all Agents.
- Weigh agent reputation and confidence scores.
- Calculate overall confidence.
- Resolve disputes in the Debate Chamber.
- Finalize the Truth Score.
